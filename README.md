# machine-learning-salud-knime
Desarrollo de modelos predictivos de aprendizaje supervisado y no supervisado para la clasificación de diagnósticos médicos.
##  Descripción del Proyecto
Desarrollo de modelos predictivos de aprendizaje supervisado y no supervisado para la clasificación de diagnósticos médicos. Se trabajó con un dataset de pacientes para predecir enfermedades pulmonares en función de variables clínicas.

##  Tecnologías y Algoritmos
* **KNIME:** Plataforma principal para el flujo de datos (ETL y Modelado).
* **Modelos Supervisados:** Random Forest, SVM (Support Vector Machine), KNN, Árboles de Decisión.
* **Clustering (No Supervisado):** K-Means, DBSCAN y Clustering Jerárquico.

##  Resultados y Conclusiones
* **Optimización:** Ajuste de parámetros (epsilon en DBSCAN, k en KNN) para mejorar la precisión del modelo.
* **ETL:** Limpieza de datos, manejo de valores nulos y normalización de variables.
* **Comparativa:** Análisis de rendimiento entre distintos algoritmos para seleccionar el modelo con mejor capacidad de generalización.
